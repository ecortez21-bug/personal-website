# **App Name**: Archisustain

## Core Features:

- Smooth Scroll Navigation: Display a fixed navigation bar with smooth scrolling to different sections: Home, About, Portfolio, Resume, and Contact.
- Portfolio Showcase: Showcase projects in a responsive grid layout, with a modal popup for detailed information and image galleries.
- Interactive Resume: Present experience and education details with a vertical timeline indicator for easy readability.
- Contact Section: Include a contact form for user inquiries (front-end only) and social media links to LinkedIn and Behance.

## Style Guidelines:

- Primary color: A deep, earthy green (#386641) to reflect sustainability and nature.
- Background color: Light beige (#F5F5DC), providing a clean and neutral backdrop.
- Accent color: Soft brown (#A68A64) for interactive elements, complementing the natural palette.
- Body and headline font: 'Alegreya', a serif font lending an elegant, intellectual, and contemporary feel, for a balance of readability and sophistication.
- Use simple, minimalist icons to represent skills and social media links.
- Employ a clean, modern layout with clear sections and ample white space.
- Incorporate subtle hover effects and smooth scrolling for enhanced user engagement.
